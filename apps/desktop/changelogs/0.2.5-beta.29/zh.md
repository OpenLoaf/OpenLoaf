## 0.2.5-beta.29

### ✨ 新功能

- **Board 媒体变体扩展**：新增 Volcengine 文生图变体、换脸模式切换，并更新口型驱动输入方式，让更多 SaaS 媒体能力可以直接在 Board 面板里配置。

### 🚀 改进

- **SaaS Media v3 集成**：将媒体客户端栈升级到 `@openloaf-saas/sdk` `0.1.21`，服务端媒体调用切换到 SDK 原生 v3 API，并保留网络重试行为，提升任务提交与轮询稳定性。
- **Board 生成控件**：改进变体命名和选择逻辑，使用能力接口返回的 tab 名称，并在变体较多时自动切换为更紧凑的下拉选择器。
- **媒体参数处理**：优化视频生成 prompt 传参、超分倍率归一化，以及任务结果解析逻辑，让 Board 媒体请求更贴合最新 SaaS API 协议。

### 💄 界面优化

- 打磨内联面板交互与节点 hover 行为，修复媒体面板指针事件泄漏，同时让已选中节点也能继续暴露锚点 hover 目标，连线更顺手。

### 🐛 问题修复

- 修复图片和视频面板变体标签仍依赖旧版 SDK preference 元数据的问题。
- 修复口型驱动输入链路，改为使用视频加音频，而不是过时的人像图片流程。
- 修复媒体任务轮询与聊天图片任务处理逻辑，使其兼容新版 SDK 返回结构，并支持音频/STT 任务返回文本结果。

### 📦 依赖

- 在 server、web 和共享 API 包中将 `@openloaf-saas/sdk` 从 `0.1.18` 升级到 `0.1.21`。
