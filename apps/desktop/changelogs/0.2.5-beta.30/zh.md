## 0.2.5-beta.30

### ✨ 新功能

- **能力驱动的连线节点选择器**：Board 连线插入菜单现在会基于实时 SaaS capabilities 按 image、video、audio、text 分组展示可用目标，并使用本地化图标与名称，创建节点更快更直观。
- **连线中点剪刀删除**：hover 或选中连线时，会在中点直接显示剪刀按钮，可以在画布上原地删除连接。

### 🚀 改进

- **Board 能力预加载**：进入 Board 时会预加载 image、video、audio 三类 capabilities，并在加载失败时提供重试入口，让连线模板更稳定可用。
- **连线交互打磨**：关闭连线选择器或取消草稿连线时，会自动恢复源节点选中状态，后续编辑更连贯。

### 💄 界面优化

- **磁吸锚点反馈**：图片锚点在靠近鼠标时会淡入并产生磁吸跟随，离开后带弹性回弹，连接目标更清晰也更有手感。
- **节点标题排版**：优化标签缩放和截断表现，让长标题在缩放时更易读，不再挤压节点界面。

### 🐛 问题修复

- 修复连线模板解析仍依赖旧静态 feature 映射的问题，输出目标现在会基于已注册的 capability variant 正确计算。
- 修复 Board 图片变体缺少显式 feature ID 导致部分生成入口在选择器中消失的问题。
