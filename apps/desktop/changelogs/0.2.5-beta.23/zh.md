## 新功能

- 新增异步聊天流式通道，补齐独立会话管理与端到端回归测试覆盖
- 新增 memory-save 能力，支持持久化聊天资产目录与按作用域写入记忆文件
- 新增用户级与项目级记忆管理接口，可列出、编辑并清空记忆文件

## 改进

- 改进前后端异步聊天传输、消息链路衔接与流式会话恢复逻辑
- 优化异步对话与记忆流程中的聊天文件持久化和资产目录解析
- 增强桌面端自动更新状态处理与下载后的 release notes 透传

## 界面优化

- 新增更完整的记忆设置面板，支持文件列表、内联编辑、打开目录和一键清空
- 优化 AI 对话中的消息列表渲染与异步聊天状态刷新
- 更新内存管理相关设置文案，并同步 en-US、zh-CN、zh-TW 本地化文本

## 问题修复

- 修复异步流式会话与记忆关联文件的 message store 持久化细节
- 修复 tool scope 与 memory tool 在可写路径和上下文文件保存上的边界行为
- 修复桌面端发布流程说明，明确要求在打新 beta tag 前先准备 changelog 文件
