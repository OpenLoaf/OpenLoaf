## 新功能

- 双击画布空白区域可快速创建文本节点并自动聚焦
- 复制画布后自动在新标签页中打开
- AI 读取文件工具新增 PDF 和 PowerPoint 格式检测引导

## 改进

- 画布缩略图：截图前保存/恢复视口状态，避免视图跳动
- 画布缩略图：打开画布时始终重新截取，确保预览图反映最新内容
- 关闭画布时同步截取缩略图，提升可靠性
- 自动布局：自动检测并保持对齐方式（左/居中/右、上/居中/下）
- 通过 requestAnimationFrame 批量处理自动调整大小，提升性能
- CanvasDoc 缓存元素数组，减少重复分配
- 优化 SpatialIndex 更新逻辑，cell key 不变时跳过删除/插入
- 支持按节点类型动态解析最小尺寸
- 思维导图方向控制仅在文本节点之间的连线上显示
- 画布内生成图片时直接保存到画布资产目录
- 新增 macOS Intel (x64) 交叉编译打包脚本

## 界面优化

- 重新设计关闭确认对话框，三按钮直接选择（取消/最小化/退出）
- 分组节点改为实色背景，替代虚线边框
- 分组工具栏：仅对媒体类型组显示统一尺寸按钮
- 编辑模式下允许文本选择

## 问题修复

- 修复中文 Windows 上 PowerShell 输出乱码（强制 UTF-8 编码）
- 画布二进制快照损坏时自动备份并从 JSON 回退恢复
- fs.stat 和 listImages 路由优雅处理 ENOENT 错误
- 修复 VFS 路径解析对 @[folderName] 格式的兼容
- 修复 Windows 增量更新：删除更新目录前先移除 junction 链接，防止破坏打包的依赖

## 移除

- 从文件系统右键菜单和空状态中移除"新建画布"选项

## 国际化

- 更新多个命名空间的中文、繁体中文、英文翻译
