## 新功能

- 新增覆盖 AI runtime、服务端接口与桌面设置流程的完整 MCP Server 支持
- 新增内建技能注册表与 skill-first 工具加载流程，让 agent 更稳定地发现核心能力
- 扩展任务上下文提取与任务详情体验，提供更完整的 agent 进度可见性

## 改进

- 优化 MCP 配置体验，补齐 reconnect-all、JSON 粘贴导入、更智能的 scope 选择和更清晰的项目名称展示
- 提升聊天链路稳定性，改进消息链拼接、task-report 同步、prompt 组装与 MCP 会话引导
- 统一工具结果可见性规则，让成功的隐藏工具继续折叠，同时保留可展示、进行中和失败状态的工具输出

## 界面优化

- 新增独立的 MCP 设置面板与服务创建弹窗，并补齐本地化文案
- 刷新任务详情和 AI 消息展示，提升状态、操作与工具输出的可读性
- 打磨 MCP 弹窗布局、滚动条表现和小屏下的溢出处理

## 问题修复

- 修复前后端聊天链路中的 task-report 持久化、消息计数和任务状态缓存对齐问题
- 修复 MCP 激活流程，支持已配置服务自动连接并减少不必要的审批打断
- 修复聊天消息中 reasoning 区块和空工具容器的渲染边界问题
