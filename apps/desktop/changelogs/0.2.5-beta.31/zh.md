## 0.2.5-beta.31

### ✨ 新功能

- **订阅中心与计费入口**：新增独立的订阅设置面板，可查看当前套餐、计费周期、积分使用量与交易记录，并支持从侧边栏账号菜单快速进入升级流程。
- **内嵌升级与充值弹窗**：OpenLoaf 现在可在原生弹窗中打开 SaaS 定价与充值流程，支持 token 刷新、超时兜底和支付成功后的状态回刷。
- **项目功能开关**：项目设置现在可以集中启用或关闭首页、历史、画布和定时功能。
- **Board 多输入槽位支持**：统一槽位系统现在支持多媒体项槽位、槽位恢复，以及更完整的关联引用添加/移除流程。
- **视频生成支持音频输入**：Qwen 视频生成面板新增可选音频输入，并保留手动上传音频作为兜底路径。

### 🚀 改进

- **项目创建流程**：新建项目时可根据配置的存储根目录自动创建文件夹路径，并提供路径预览与快速打开入口。
- **项目设置导航**：项目设置左侧菜单现在会展示当前项目图标和标题，编辑时上下文更清晰。
- **Desktop 更新说明可见性**：关于页的 changelog 加载现在会同时展示当前 desktop、server 和 web 版本。
- **订阅嵌入稳定性**：计费嵌入在 hydration 竞争场景下会重试 auth postMessage，并采用更宽的横向弹窗布局以提升加载成功率。

### 💄 界面优化

- **Board 槽位布局打磨**：单项与多项槽位混合状态下的渲染更清晰，chip 计数和“继续添加”入口也更自然。
- **Sidebar 与设置页细节优化**：调整了侧边栏图标按钮行为、弹窗尺寸，以及项目和设置页面中危险操作按钮的对比度。
- **视频首帧更清晰**：视频播放器会在 `loadeddata` 后再移除 poster，减少播放前出现低清晰缩略图或黑帧的情况。

### 🐛 问题修复

- 修复项目目录缺失时未能平滑清理注册表和数据库的问题，前端现在会稳定收到 `PROJECT_REMOVED` 错误。
- 修复新建项目和临时项目默认未启用 `canvas` 的问题。
- 修复计费弹窗在 iframe token 或 hydration 时序竞争下卡住加载的问题。
- 修复 Board 多项媒体槽位持久化时错误折叠成单个引用、导致恢复不完整的问题。
- 修复 Qwen 视频生成参数组装遗漏音频输入的问题，连接的音频现在会正确传入 `inputs.audio`。
