## 更新内容

### ✨ 技能市场
- 浏览、安装和管理技能市场中的技能
- 可更新技能通知，支持逐个更新或一键全部更新
- 技能详情弹窗，显示评分和描述
- 完整多语言支持（中文简体、中文繁体、英文、日文）

### 🚀 上下文窗口管理
- 更保守的 token 估算策略，防止 API 400 错误
- 三段式裁剪策略：启发式压缩 → 渐进丢弃 → 硬尾部保留
- 正确估算图片、文件和工具调用部分的 token 数量
- 自动压缩在摘要失败或无模型时回退到硬裁剪

### 🔒 安全
- 防止技能导入时的 zip 路径穿越攻击
- 清洗技能导入临时文件名
- 技能归档导入增加 64MB 大小限制

### 🐛 问题修复
- 技能翻译失败时显示更友好的错误提示（未配置模型、网络错误、云端配额不足等）

### 💄 界面优化
- 浏览器面板底部圆角增大
- 浮动面板去掉边框和阴影，更简洁
- 反馈对话框使用现代 `userAgentData` API 检测平台

### 📦 依赖更新
- 升级 @openloaf-saas/sdk 至 v0.1.34
