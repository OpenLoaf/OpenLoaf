## 0.2.5-beta.27

### ✨ 新功能

- **Board Anchor Slot 工作流**：新增类型化输入/输出槽位、动态节点模板、分组插入器、实时连线校验，以及文本 `@` 引用，让媒体节点和提示词节点的连接方式更自然。
- **媒体生成能力扩展**：新增多种图片、视频、音频生成变体，媒体生成链路迁移到更新的 SaaS Media 接口，并重构变体架构以支持更丰富的能力专属输入。
- **图片内联调整**：为图片节点新增全屏调整层，支持裁剪、旋转和翻转。

### 🚀 改进

- **媒体上传与播放链路**：优化 Board 上传流程、媒体任务编排、远程视频接入和视频流播放，让生成内容和下载内容在应用内更稳定。
- **Board 编辑体验**：改进媒体面板、选择与插入交互、工具栏、版本栈 hooks 和分组节点选择器，提升画布操作效率。
- **聊天与文件体验**：打磨聊天工具渲染、附件处理、会话行为，以及视频/文件预览流程，使其更好适配新的媒体链路。

### 💄 界面优化

- 统一刷新侧边栏、项目/文件视图、设置、任务、桌面 widget 与加载态等多个桌面端界面区域，整体体验更一致。

### 🌐 国际化

- 为新的 Board 媒体面板、输入槽位系统、图片调整控件以及相关 AI/聊天界面补充并更新多语言文案。

### 🐛 问题修复

- 修复 Board 媒体重构过程中暴露出的媒体播放、上游数据流、变体标签与翻译键不一致等问题。
- 修复新媒体下载链路与图片调整路径中的合并后类型问题。

### 📦 依赖

- 升级 `@openloaf-saas/sdk` 集成到更新的媒体 API 代际，支撑新的 Board 工作流。
