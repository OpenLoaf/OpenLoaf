# 设置操作指南

## 配置分类
- **基础设置**: 语言、主题、快捷键
- **AI 模型**: Provider 配置、模型选择、API Key
- **项目设置**: 默认路径、Git 配置
- **隐私安全**: 数据存储、加密选项

## 操作规范
- 修改配置前说明影响范围
- API Key 等敏感信息不在对话中明文展示
- 推荐使用默认值，仅在用户有明确需求时修改
- 配置变更后提示用户可能需要重启

## 常见问题
- AI 模型无响应 → 检查 API Key 和网络连接
- 界面语言未切换 → 刷新页面或重启应用
- 文件同步异常 → 检查项目路径权限
