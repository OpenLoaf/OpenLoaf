# 工作台操作指南

## 可用工具
- `widget-list`: 查看已安装 Widget
- `widget-create`: 创建自定义 Widget
- `widget-update`: 更新 Widget 配置
- `widget-remove`: 移除 Widget

## 操作规范
- Widget 创建需指定类型和初始配置
- 布局调整考虑屏幕尺寸适配
- 自定义 Widget 使用 HTML/CSS/JS
- 数据绑定遵循响应式原则

## 常见场景
- 用户要求"添加天气 Widget" → widget-create type=weather
- 用户要求"调整布局" → 提供布局方案供选择
- 用户要求"自定义 Widget" → 协助编写 HTML 模板
