# Node Layout Spec

每个画布节点由四部分组成：**顶部工具栏** + **节点主体** + **底部 AI 面板** + **左右锚点**。本文档定义各节点类型的完整布局。

## 通用约定

### 工具栏 (Selection Toolbar)
- 选中节点时显示，位于节点正上方居中
- **黑白风格**：`text-foreground` / `text-muted-foreground`，hover `bg-foreground/8`
- 唯一例外：删除按钮 `text-destructive`
- 版本导航 `[← 1/4 →]` 仅在有版本栈时显示
- 按钮之间用 `┃` 分隔符分组

### AI 面板
- 宽度 420px，圆角 `rounded-xl`，`shadow-lg bg-card`
- 位于节点下方 8px（PANEL_GAP_PX）
- 缩放 `scale(1/zoom)` 保持固定屏幕尺寸
- 通过 Portal 渲染到 panelOverlay 层

### 锚点
- 左侧锚点 = 输入端（接收上游连线）
- 右侧锚点 = 输出端（拖出创建下游节点）
- 锚点图标 `⊕`，hover 时显示可创建的节点类型列表（connectorTemplates）

### AI 面板布局

**模式 Tab**：`[文生图]` `[图生图]`
- 文生图：prompt 必填，图片插槽可选（风格参考）
- 图生图：图片插槽必填（原图），prompt 可选

**上游插槽区**：
- 预设风格按钮（52×52，圆角）+ 上游图片缩略图（52×52，右上角连接数徽章）
- 右上角 `[⤢]` 展开按钮（全屏查看上游素材）
- 手动添加 `⊕` 按钮
- 超限时显示 "已选 2/10" + 每张图带 checkbox 勾选

**Prompt 区**：
- Textarea，min-height 72px
- 右下角 `[✨]` 增强按钮（Wand2 图标）
- 上游 Text 自动填入时显示 "⚡上游已加载" 横幅

**参数区**：
- 比例按钮组：`[1:1] [16:9] [9:16] [4:3]`，pill 样式
- 清晰度按钮组：`[1K] [2K] [4K]`，pill 样式
- 高级设置折叠区：反向提示词、种子值等

**底栏**（紧凑模式，与参数区二选一显示）：
- 左侧：`[M 模型名 ▾]` `[比例·清晰度 ▾]`
- 右侧：`[文A翻译]` `[⚙设置]` `[⚡积分]` `[×N数量▾]` `[⬆生成▾]`

### 推荐按钮 (节点底部)

| 按钮 | 图标 | 颜色 | 显示条件 | 动作 |
|------|------|------|---------|------|
| 文生图 | Sparkles | ol-blue | 始终显示 | deriveNode → ImageNode (下游) |
| 图片反推 | ImagePlus | ol-blue | 文本为空 + 无上游图片 | deriveNode → ImageNode (上游) |
| 文生视频 | Video | ol-purple | 始终显示 | deriveNode → VideoNode (下游) |
| 文字转语音 | Volume2 | ol-green | 始终显示 | deriveNode → AudioNode (下游) |

- 推荐按钮使用**语义色**（内容层），不用黑白
- 容器样式 `ol-glass-toolbar`（玻璃磨砂背景）
- 位置 `top-full left-1/2 translateX(-50%)`，节点底部居中
- 缩放跟随 `--label-scale` CSS 变量
- 全局隐藏：节点锁定/只读时不显示；仅选中时渲染

### AI Panel (仅文本→文本下游节点)

普通 TextNode 无 AI 面板。只有作为 AI 润色/翻译/改写的下游节点时才显示：
- 上游文本预览区（只读，灰色背景）
- AI 指令输入区
- 底栏：`[M 模型▾]` `[文A翻译]` `[⚡积分]` `[⬆发送]`
- 无数量选择、无堆叠/覆盖切换

### 锚点 connectorTemplates

| 拖出方向 | 目标节点 | 传递内容 |
|---------|---------|---------|
| → 右侧 | ImageNode | 文本 → prompt (文生图) |
| → 右侧 | VideoNode | 文本 → prompt (文生视频) |
| → 右侧 | AudioNode | 文本 → 待合成文本 (TTS) |
| → 右侧 | TextNode | 文本 → AI 润色/翻译/改写 |

### 节点尺寸

| 属性 | 值 |
|------|-----|
| Min | 200 × ~54 px (fontSize + padding) |
| Max | 720 × 420 px |
| Default | 200 × 200 px (sticky 样式) |
| Resize | 自由 |

### 工具栏按钮

| 按钮 | 图标 | 显示条件 | 动作 |
|------|------|---------|------|
| 分支 | GitBranch | 始终 | 从当前 turn 分叉出新对话方向 |
| 折叠 | ChevronsLeftRight | 仅 root turn | 折叠整个 session |
| 删除 | Trash2 | 始终 | 删除此 turn（自动重连跳过） |

### 锚点

| 位置 | 类型 | 连线语义 | 说明 |
|------|------|---------|------|
| 左侧中央 | chat-in | chat-flow | 自动创建，不可手动操作 |
| 右侧中央 | chat-out | chat-flow | 自动创建，不可手动操作 |
| 右侧 75% | resource-out | data-flow | 仅 hasResources 时显示，用户可拖出 |

**resource-out 拖出规则**：
- 拖出 = 一根线携带该 turn 的全部 resources
- 目标节点 AI 面板自动加载资源
- 超限时复用"上游输入超限"规则：面板中 checkbox 勾选
- 拖出的 data-flow 与普通连线行为一致（一次性拷贝、stale 提示）

### 无 AI 面板

ChatTurn 没有独立的 AI 面板。对话输入内嵌在**最后一个 turn 底部**的继续输入框中。

## AI 面板底栏统一规范

所有媒体节点（Image/Video/Audio）的 AI 面板底栏遵循相同布局：

```
┌──────────────────────────────────────────────────────────────┐
│ [M 模型名 ▾]  [参数摘要 ▾]       [文A] [⚙] [⚡积分] [×N▾] [⬆▾] │
│  ↑模型选择      ↑比例·清晰度·时长   ↑翻译  ↑高级  ↑预估  ↑数量  ↑生成 │
└──────────────────────────────────────────────────────────────┘
```

| 元素 | Image | Video | Audio | Text→Text |
|------|:---:|:---:|:---:|:---:|
| 模型选择器 | ✅ | ✅ | ✅ | ✅ |
| 参数摘要 | 比例·清晰度 | 质量·比例·时长 | 时长 | - |
| 翻译 (文A) | ✅ | ✅ | ✅ | ✅ |
| 设置 (⚙) | ✅ (高级参数) | - | - | - |
| 积分 (⚡) | ✅ | ✅ | ✅ | ✅ |
| 数量 (×N) | ✅ (1-15) | ✅ (1-4) | - | - |
| 生成按钮 (⬆) | ✅ 带▾下拉 | ✅ 带▾下拉 | ✅ 带▾下拉 | ✅ 无下拉 |

**生成按钮下拉**（媒体节点）：
- 堆叠模式（默认）：正常生成，结果入版本栈
- 覆盖模式：第1个结果覆盖 primary，其余堆叠

## 提示词增强 (Prompt Enhancement)

所有含 prompt 输入的 AI 面板统一提供增强功能：

```
┌ Prompt ───────────────────────────────── [✨] ─┐
│ 用户输入的 prompt 文本...                        │
│                                    增强按钮(✨) → │
└──────────────────────────────────────────────────┘
         ↓ 点击增强后
┌ Prompt (增强预览) ────────────────────────────────┐
│ ┌─ bg-ol-purple/3 border-ol-purple/20 ─────────┐ │
│ │ [Enhanced] 增强后的 prompt 文本...              │ │
│ └───────────────────────────────────────────────┘ │
│ [✓确认]  [↻再增强]  [✗取消]                        │
└──────────────────────────────────────────────────┘
```

- 按钮位置：textarea 右下角，`Wand2` 图标
- 按钮样式：`text-muted-foreground hover:bg-foreground/8`（黑白风格）
- 加载态：`Loader2 animate-spin` + textarea shimmer 遮罩
- 预览态：`bg-ol-purple/3 border-ol-purple/20` + "Enhanced" 徽章
- 操作栏：[确认] [再增强] [取消]
- 走独立 LLM 通道，不占媒体任务队列
- 3 套增强 prompt 模板：图片(补画面细节) / 视频(补动态描述) / 音频(补风格描述)
